📖 使用说明
请仔细阅读以下指南，确保顺利使用本工具

1️⃣ 运行环境要求 重要

本工具必须在 Windows 系统上运行，且需要安装 Microsoft Word 2016 或更高版本。 不支持 macOS / Linux，也不支持 WPS Office。

2️⃣ 启动服务

双击 start_server.bat 即可启动服务， 浏览器会自动打开 http://127.0.0.1:5000。 如果浏览器未自动打开，请手动访问该地址。

3️⃣ 上传文件

点击「选择文件」按钮或直接拖拽文件到虚线区域内。 支持 PDF（.pdf）和 Word（.doc / .docx）格式，单个文件最大 50MB。 上传后会自动开始转换。

4️⃣ 查看进度与下载

上传后页面底部会显示实时转换进度。转换完成后会出现绿色的 「下载」按钮，点击即可保存到本地。

5️⃣ 文件安全 隐私

所有转换在本地计算机上完成，文件不会上传到任何云端服务器。 转换完成后的临时文件存储在 conversions 文件夹中， 可随时通过「打开临时文件夹」按钮查看。

6️⃣ 自动清理机制

系统会在每次访问首页时自动清理以下文件：
• 超过 1 小时的旧临时文件
• 每个文件夹超过 10 个时，删除最早的文件
也可使用下方「清空所有临时文件」按钮手动清理。

7️⃣ Word 进程占用 注意

转换过程中会调用 Microsoft Word 后台进程。如果转换失败，请检查： 任务管理器中是否有 WINWORD.EXE 进程未关闭， 手动结束该进程后重试。转换超时时间为 120 秒。

8️⃣ 常见问题排查

转换失败？检查文件是否损坏、Word 是否正常安装。
无法启动？确保已安装 Python 依赖，运行 pip install -r requirements.txt。
端口被占用？编辑 app.py 修改端口号。
